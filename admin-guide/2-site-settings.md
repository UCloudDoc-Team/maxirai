# 站点配置

在首次使用本系统时，需先完成站点的基础配置。站点配置包括网站基本信息、界面语言以及管理员密码管理。

## 站点配置

站点配置即为网站的基本信息设置，包括站点名称与 Logo 上传。

1.  使用超级管理员账号登录管理端控制台。
    
2.  点击页面左下角账号名称右侧的展开按钮，选择“站点管理”。
    
    ![](images/101.jpg)
    
3.  输入站点名称，上传网站的 Logo，上传登录页面背景图。
    
    ![](images/102.jpg)
    
    Logo 支持上传 3 种类型：
    
    - **Logo**：用于正常模式登录页面，建议尺寸为 140px * 40px。
        
    - **Dark Logo**：用于暗黑模式登录页面，建议尺寸为 140px * 40px。
        
    - **Mini Logo**：用于导航栏，建议尺寸为 40px * 40px。
        
4.  点击“保存”。

## 切换界面语言

你可以自定义界面显示语言，目前支持**中文**和**英文**。

1.  使用超级管理员账号登录管理端控制台。
    
2.  点击页面左下角账号名称右侧的展开按钮，选择“语言” > _目标语言_。

![](images/103.jpg)

## 修改系统管理员密码

系统管理员为**全局唯一**的最高权限账号。出于安全考虑，请妥善管理密码。

?> 如系统管理员密码丢失，请联系技术支持。

操作步骤：

1.  使用超级管理员账号登录管理端控制台。
    
2.  点击页面左下角账号名称右侧的展开按钮，选择“修改密码”。
    
    ![](images/104.jpg)
    
3.  在弹出来的对话框中，输入新密码，确认新密码，并点击“确认修改”。
