# 团队管理

在一个组织内可创建多个团队，各团队间数据相互隔离，确保信息安全；而团队内部成员则可通过团队空间实现数据共享，提升协作效率。

## 创建团队

1.  在左侧导航栏中，选择“团队管理”。在“团队管理”页面，点击右上角“创建团队”按钮。
    
    ![](images/114.jpg)
    
2.  在“添加团队”页面，配置团队信息。
    
    ![](images/115.jpg)
    
    关于配置项的详细说明，请参考下表：
    
    | **参数** | **是否必填** | **说明** |
    | :- | :- | :- |
    | 团队名称 | 是 | 团队的名称。 |
    | 团队管理员 | 是 | 指定团队管理员，可设置多个。<br/>团队管理员可在用户控制台上管理团队空间，包括团队成员、文件、知识库/数据集、以及 Agent 等。详细操作，请参考[管理员指南](admin-guide/1-overview.md)。 |
    | 聊天模型 | 是 | 用于对话生成的默认模型。 |
    | 嵌入模型 | 是 | 用于对文档和数据进行 Embedding 的默认模型。 |
    | Rerank 模型 | 是 | 用于重排序的模型。 |
    | NL2SQL 模型 | 是 | 用于自然语言转 SQL 代码的模型。 |
    | NL2Python 模型 | 是 | 用于自然语言转 Python 代码的模型。 |
    
3.  完成配置后，点击“创建团队”。
    

## 查看团队信息

1.  在团队列表中，找到目标团队，点击“操作”列中的查看团队信息按钮。
    
    ![](images/116.jpg)
    
2.  在团队详情页面，可以查看团队的名称、管理员信息、成员数量及列表以及团队的创建时间。
    

## 修改团队信息

1.  在团队列表中，找到目标团队，点击“操作”列中的编辑按钮。
    
2.  在“编辑团队”页面中，按需修改团队名称或团队管理员，点击“保存修改”。
    
    ![](images/117.jpg)
    
    - 该操作支持修改团队的名称，以及删除或添加团队管理员。
        
    - 原有团队管理员删除后，对应用户会降级为团队成员，不会被移出团队空间。
        
    - 完成团队信息修改后，为避免使用过程中的混淆或误操作，请及时将更新内容同步给相关团队管理员及成员。
        
## 删除团队

!> 团队删除后，对应团队空间将无法使用，请谨慎操作。

1.  在团队列表中，找到目标团队，点击“操作”列中的删除按钮。
    
2.  在弹出的确认对话框中，确认即将删除的团队为目标团队，点击“删除”按钮。
