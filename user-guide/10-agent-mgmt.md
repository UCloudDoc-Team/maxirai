# Agent 管理

在 MAXIR AI 中，所有问答交互均由 Agent 执行。要使用系统进行问答，必须先创建并配置 Agent。本章节将介绍 Agent 的作用及其创建与管理方式，帮助用户快速构建可用的智能问答代理。

## 创建 Agent

1.  登录用户控制台。
    
2.  点击账号信息右侧的展开按钮，选择想要进入的空间。
    
    ?>如需创建团队 Agent，请确保：
    ?>- 登录账号为目标团队的管理员。
    ?>- 已切换到目标团队空间。      

    ![](images/035.jpg)

3.  在左侧导航栏中，点击“添加 Agent”。
    
4.  在“聊天配置”页面，完成“资源类型”、“基础设置”、“提示引擎”以及“模型设置”的相关配置。
    
    ![](images/036.jpg)

    资源类型相关的参数说明：
    | **配置项** | **说明** |
    | :- | :- |
    | 知识库 | 选择 Agent 用于问答的知识库。<br/>如无可用的知识库，请参考“创建知识库”章节完成知识库创建。 |
    | 数据集 | 选择 Agent 用于问答的数据集。<br/>如无可用的数据集，请参考“创建数据集”章节完成数据集创建。 |
    | 无 | 不指定任何知识库/数据集，直接使用通用大模型进行问答。 |

    基础设置相关的参数说明：

    | **配置项** | **说明** |
    | :- | :- |
    | Agent 名称 | Agent 的名称。 |
    | Agent 描述 | Agent 用途/功能的简要描述。 |
    | 空回复 | 当用户输入问题超出 Agent 回复范围时，返回的回复，例如“抱歉，我无法回答这个问题”。 |
    | 问候语 | 新开一个 Session 时，Agent 默认发出的第一条消息。例如“你好！我是你的助手，有什么可以帮到你的吗？”。 |
    | Agent 权限 | Agent 的权限范围，可选项包括：<br/>-   **个人**：仅个人可以使用该 Agent。<br/>-   **团队**：团队内的所有成员均可以使用该Agent。仅当登录账号为团队空间管理员且所在空间为团队空间时，可选。<br/>**说明**：仅当登录账号为团队管理员时，有该配置项。 |
    | 历史上下文 | 用于问答上下文的历史消息数量。 |
    | 知识库召回语言 | 知识库支持的召回语言。目前支持选择“中文”和“英文”，可多选。设置后，Agent 可使用多种语言对知识库的内容进行召回。建议在用户提问语言和数据库内容语言不一致时配置，例如，用户 A 用中文对内容为英文的知识库提问时，推荐同时选择 `中文` 和 `English` 选项。<br/>**说明**：如提问语言和知识库内容语言一致时，配置此项可能造成召回内容失真，**建议不要配置此项**。 |
    | 回复语言 | Agent 回复的语言。 |
    | Web Search 配置 | Agent 使用的网络搜索配置。<br/>**说明**：仅当资源类型为“知识库”时，该参数可用。 |

    提示引擎相关的参数说明：

    | **配置项** | **说明** |
    | :- | :- |
    | 系统提示词 | 系统提示词，用于指导 AI Agent 的角色定位、语气风格、行为约束。 |
    | 总结提示词 | 用于指导 Agent 返回总结内容的提示词，避免 Agent 返回冗余内容。例如：“只返回结论，不需要提供洞察。”<br/>**说明**：仅当资源类型为“数据集”时，该参数可用。 |
    | 相似度阈值 | 相似度阈值是用来判断两个文本是否“足够相似”的一个数值标准，取值范围为 0 ～ 1。当系统计算出用户输入与候选项之间的相似度分数后，若该分数达到或超过设定的阈值，即视为“匹配”或“相关”。<br/>**说明**：仅当资源类型为“知识库”时，该参数可用。 |
    | 关键词权重 | 关键词相似度权重是用于衡量特定关键词在相似度计算中所占比重的参数。它用于引导系统在判断两个文本之间的相似程度时，**更加关注某些关键字或短语**的匹配情况，从而提升语义识别的准确性和针对性。<br/>**说明**：仅当资源类型为“知识库”时，该参数可用。 |
    | Top N | 表示从一组候选结果中按相关性或评分排序后，返回排名前 N 个结果。<br/>**说明**：仅当资源类型为“知识库”时，该参数可用。 |


    模型设置相关的参数说明：

    | **配置项** | **说明** |
    | :- | :- |
    | 模型 | 用于聊天的大语言模型。 |
    | 温度 | 控制生成词汇的概率分布的平滑度。取值范围为 0~1。较低值意味着输出更稳定、确定性强，较高值意味着更随机、创造性更强。 |
    | Top P | 控制语言模型每次生成时**从累计概率达到阈值 p 的词汇集合中进行采样**，以平衡输出的合理性与多样性的一种采样策略参数。 |

5. 点击“确定”。
    
## 使用 Agent 进行问答

Agent 创建完成后，即可使用该 Agent 进行问答。

1.  在“我的 Agent”或者“团队 Agent”区域，找到目标 Agent。点击 Agent 名称右侧的添加按钮，开始一个新会话。
    
    ![](images/037.jpg)
    
2.  在新会话页面底部的对话框中，输入你想要问的问题，点击“发送”。
    

## 修改 Agent 信息

1.  在“我的 Agent”或者“团队 Agent”区域，找到你想要修改的 Agent。
    
2.  点击 Agent 名称右侧的更多按钮，选择“编辑 Agent”。
    
    关于各参数的说明，请参考“创建 Agent”章节里的参数列表。
    
    ![](images/038.jpg)
    
3.  点击“确定”。
    

## 查看 Agent 信息

1.  在“我的 Agent”或者“团队 Agent”区域，找到目标 Agent。
    
2.  点击该 Agent 名称，进入 Agent 详情页，即可查看该 Agent 配置详情。 

    ![](images/039.jpg)

Agent 详情页还提供编辑和删除入口，如需编辑 Agent 信息或删除该 Agent，可点击对应按钮完成相应操作。

## 删除 Agent 中的 Session

1.  在“我的 Agent”或者“团队 Agent”区域，找到你想要删除的 Agent。
    
2.  点击 Agent 名称右侧的更多按钮，选择“批量删除 Session” 或者“清空 Session”。
    
    ![](images/039.5.jpg)
    
3.  （可选）如选择“批量删除 Session”，在 Session 列表中，勾选你想要删除的 Session，点击“删除”按钮。
    
4.  在弹出的删除确认框中，点击确认删除操作。
    

## 删除 Agent

1.  在“我的 Agent”或者“团队 Agent”区域，找到你想要删除的 Agent。
    
2.  点击 Agent 名称右侧的更多按钮，选择“删除 Agent”。
    
    ![](images/040.jpg)
    
3.  在删除确认对话框中，确认删除信息无误后，点击“确认删除”。
