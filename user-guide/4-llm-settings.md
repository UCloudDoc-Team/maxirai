# 模型配置

在 “模型管理” 页面，你可以为当前空间灵活设置所使用的模型，包含聊天、嵌入、重排序、NL2SQL、NL2Python 等模型，以满足使用场景的问答需求。

不同工作空间所采用的模型配置也不相同：

- 团队空间：继承组织管理员在创建团队时设置的默认配置，仅**团队管理员**有权限在团队空间内进行修改。
    
- 个人空间：在你首次登录你的个人空间时，系统会默认弹窗提醒你进行初始设置。
    

## 修改模型默认配置

1. 在左侧导航栏中，选择“模型管理”。
    
2. 在“设置默认模型”对话框中，完成模型配置。

    ![](images/001.jpg)

    支持的配置项说明如下表所示：

    | **配置项** | **说明** |
    | :- | :- |
    | 聊天模型 | 用于对话生成的默认模型。 |
    | 嵌入模型 | 用于对文档和数据进行 Embedding 的默认模型。 |
    | 重排序模型 | 用于重排序的模型。 |
    | NL2SQL 模型 | 用于自然语言转 SQL 代码的模型。 |
    | NL2Python 模型 | 用于自然语言转 Python 代码的模型。 |
