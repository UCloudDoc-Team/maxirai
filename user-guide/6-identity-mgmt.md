# 身份管理

MAXIR AI 支持对团队成员进行身份管理，可通过身份模板为不同用户创建不同的身份。用户身份与数据库数据源的业务角色相关联，用户对指定数据库的访问权限受其匹配的业务角色限制，从而实现更细粒度的权限控制。

## 创建身份模板

1. 在左侧导航栏中，选择“身份模板”。在“身份模板”页面，点击右上角的“+创建身份模板”按钮。
    
    ![](images/01-5.jpg)
    
2. 在“创建身份模板”页面，填写身份模板的名称和描述，添加身份标识。

    ![](images/02-5.jpg)

    参数说明：

    | **参数** | **说明** |
    | :- | :- |
    | 名称 | 身份模板的名称。 |
    | 描述 | 身份模板的描述。 |
    | 身份标识 | 身份标识，用于从不同的维度来确定用户的身份，包含：<br/>- **名称**：身份标识的名称，例如“区域”。<br/>- **描述**：身份标识的描述。<br/>- **编码**：身份标识的编码。<br/>- **说明**：可指定多个身份标识。 |

## 编辑身份模板

1.  在“身份模板”页面，找到目标身份模板，点击“操作”列的“编辑”按钮。
    
2.  完成信息修改，点击“保存”。
    

## 删除身份模板

如下情况时，身份模板不能删除：

- 存在基于该身份模板创建的身份实例
    
- 存在与该身份实例绑定的业务角色
    
操作步骤：

1. 在“身份模板”页面，找到目标身份模板，点击“操作”列的“删除”按钮。
    
2. 在弹出的确认对话框中，确认要删除的身份模板信息，点击“确定”。
