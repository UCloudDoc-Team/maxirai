# 团队管理

团队管理功能仅对**团队管理员**开放。在执行相关操作前，请确保：

- 登录账号拥有目标团队的管理员角色权限
    
- 工作空间已切换至目标团队的团队空间
    

## 添加成员

1.  在左侧导航栏中，选择“团队管理”。点击页面右上角的“添加成员”按钮。
    
    ![](images/002.jpg)
    
2.  在“添加团队成员”页面，选择需要添加的成员名单，点击“添加成员”。

    ![](images/136.jpg)

## 为成员添加身份

团队成员的**身份**与其在访问数据库数据集时的**访问角色**相绑定。访问角色定义了成员在访问数据库数据集时所具备的一组权限，包括可访问的数据范围及可执行的操作类型。

> 如需了解访问角色相关信息，请参考[创建数据库数据源](/maxirai/user-guide/7-data-source-mgmt.md#创建数据库数据源)章节。

1.  在“团队成员”页面，找到目标成员所在行，点击“操作”列的“查看身份实例”按钮。
    
    ![](images/004.jpg)
    
2.  在“查看身份实例”页面，点击“添加身份实例”。
    
    ![](images/03-5.jpg)
    
3.  完成身份实例配置，点击“保存”。
    
    ![](images/04-5.jpg)
    
    参数说明如下表所示：
    
    | **参数** | **说明** |
    | :- | :- |
    | 业务身份模板 | 指定用于创建身份实例的模板。 |
    | 名称 | 身份实例的名称。 |
    | 描述 | 身份实例的描述。 |
    | 业务身份标识 | 指定业务身份标识的具体值。 |
    
4.  如需创建多个身份实例，重复步骤 2、3 即可。
    

## 删除成员

1.  在“团队管理”页面，找到需要删除的成员信息，点击对应“操作”列的删除按钮。
    
    ![](images/003.jpg)
    
2.  在弹出的确认信息中，确认即将删除的成员为目标成员，点击“删除”。  

    > 仅可删除团队成员，团队管理员**不支持**直接删除。
