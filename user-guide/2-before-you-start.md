# 开始之前

在正式使用 **MAXIR AI 用户端控制台**之前，你需要先了解系统中的空间类型以及团队内的角色说明。这些概念将帮助你快速熟悉系统结构，并在实际操作中更高效地完成任务。

## 工作空间说明

MAXIR AI 为用户提供两类工作空间，以满足不同的使用需求：

- **团队空间**

    - 团队范围内的共享工作空间。
        
    - 团队空间中的 **Agent、知识库/数据集、模型配置**可由所有团队成员共同使用。
        
    - 团队空间中的 Agent 分为个人 Agent 和团队 Agent，个人 Agent 只能创建者自己使用。
        
    - 每个团队的空间彼此独立，互不影响。
        
    - 团队管理员负责维护该空间的配置与资源。
        
- **个人空间**
    
    - 每个用户仅有**一个个人空间**，且资源完全独立，不与团队共享。
        
    - 个人空间中的文件、知识库/数据集和 Agent 完全归属个人所有，仅本人可见。
        
- **工作空间选择的重要性**
    
    - 用户在使用控制台时，需明确选择**团队空间**或**个人空间**。
        
    - 不同空间下可访问的资源和权限不同，操作对象也完全隔离。
        
    - 一个用户可以：
        
        - 拥有一个个人空间；
            
        - 被添加到多个团队；
            
        - 并在多个团队中担任管理员。
            
## 团队角色说明

在用户端控制台中，主要涉及以下两类角色：

- **团队管理员**

    - 拥有团队空间的管理权限。
        
    - 可以修改团队空间内的 **模型配置**。
        
    - 可以管理团队空间内的 **文件、知识库、数据集等资源**。
        
    - 可以管理团队空间内的 **Agent**，以及使团队 Agent 进行问答。
        
- **团队普通成员**

    - 作为团队的普通使用者，在团队空间内只能：
    
    - 使用团队 Agent 进行问答操作
        
    - 创建个人 Agent 进行问答操作
        

## 空间类型、角色与权限映射表

| **空间类型** | **角色** | **权限说明** |
| :- | :- | :- |
| 团队空间 | 管理员 | - 管理团队空间内的**模型配置**    <br/>- 管理团队空间内的**文件、知识库/数据集** <br/>- 管理团队 **Agent**（创建/修改/删除）<br/>- 使用团队 Agent 进行问答 |
|  | 普通成员 | - 使用团队 Agent 进行问答操作 <br/>- 创建个人 Agent 进行问答操作 |
| 个人空间 | 用户本人 | - 每个用户仅有 **一个个人空间** <br/>- 管理并使用个人空间内的 **文件、知识库/数据集、Agent** <br/>- 资源完全归个人所有，仅本人可见，不与团队共享 |
