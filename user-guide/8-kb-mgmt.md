# 知识库管理

知识库是一组在用途或内容上高度相关的非结构化文件集合，为大语言模型提供精准且具上下文的知识支持，从而实现高质量、可信赖的智能问答体验。本节将介绍如何对知识库进行有效的组织、配置与维护，助力构建结构清晰、可控性强且高效运行的知识体系。

## 创建知识库

如需使用上传的文件进行问答，首先需要将文件添加至知识库中。

知识库支持的文件类型包括：**pdf、docx、doc、txt、md、json、pptx**。

1. 在左侧导航栏中，选择“知识库”。点击“添加知识库”卡片。
    
    ![](images/010.jpg)
    
2. 在“创建知识库”页面，完成知识库配置。  

    ![](images/036.jpg)
    
    关于配置参数说明，请见下表：

    | **参数** | **是否必填** | **说明** |
    | :- | :- | :- |
    | 知识库名称 | 是 | 知识库的展示名称。 |
    | 知识库描述 | 是 | 知识库内容、用途等的简要描述。 |
    | 文档默认切片方法 | 是 | 指定文档使用的默认切片方法。 |

    文档切片方法说明：

    | **对应切片方法** | **说明** |
    | :- | :- |
    | General | 采用通用的视觉检测切片方法，将连续文本自动拆分为较小的片段，适用于大多数文档类型。<br/>**支持的文件类型：** **`pdf`**、**`docx`**、**`xlsx`**、**`csv`**、**`txt`**、**`md`**、**`json`**、**`doc`**。 |
    | Q&A | 专为问答型文件设计。 <br/>- 若为 XLSX 文件，需包含两列（无表头），第一列为问题，第二列为答案，支持多 Sheet，只要列结构正确即可。 <br/>- 若为 CSV/TXT 文件，需使用 UTF-8 编码，并以 TAB 分隔问题和答案。未遵循规范的行将被忽略，每组问答将作为一个独立片段处理。<br/>**支持的文件类型：**`xlsx`**、**`csv` **、**`txt`**。** |
    | Presentation | 适用于具备分节结构的演示类文档。系统使用最高层级的章节标题作为切片标志，保留每一部分的完整内容与视觉结构。<br/>**支持的文件类型：**`pptx`**、**`pdf`**。** |
    | Laws | 针对法律法规文档进行优化切片，确保条款结构清晰。<br/>**支持的文件类型：**`pdf`**、**`docx` **、**`txt`**、**`doc`**。** |
    | Book | 针对内容超长的文档（如图书），进行大粒度分段，兼顾完整性与问答覆盖。<br/>**支持的文件类型：**`pdf`**、**`docx` **、**`txt`**、**`doc`**。** |
    | Manual | 适用于具备分层结构的手册类文档。系统以最高层级的标题作为切片依据，确保同一部分中的正文与图表保持完整，块的粒度较大。<br/>**支持的文件类型：**`pdf`**、**`docx`**。** |
    | Paper | 根据标题结构（如摘要、1.1、1.2 等）进行切片，使内容更易于概括与理解，适合获取段落层级的深入问答。<br/>**支持的文件类型：**`pdf`**。** |

3. 配置完成后，点击“创建知识库”。
    

## 管理知识库中的文档

刚创建完成的知识库为空知识库，如需要使用对应知识进行问答，还需要为知识库添加文件。

### 添加文件

1.  在“知识库管理”页面，点击目标知识库卡片进入知识库详情页。在知识库详情页，点击右上角的“添加文件”按钮。
    
    ![](images/011.jpg)
    
2.  在“选择文件”页面，选择需要加入该知识库的文件或文件夹，点击“添加到知识库”。

    ![](images/140.jpg)
    
3.  添加完成后，在知识库详情页的文档列表中，找到新增的文件，点击“操作”列的解析按钮。
    
    > 只有**解析成功**的文件内容，才能在问答过程中被有效召回。若需使用该文件参与问答，请务必完成解析操作。
    
    ![](images/012.jpg)
    
    如需同时解析多个文档，可勾选目标文档，点击列表右上方的“批量解析”按钮。
    
4.  待文件状态从“未解析”变成“解析成功”，文件完成解析，可以被大模型召回。
    

### 修改文件切片方法

加入知识库的文件，默认继承知识库的文档切片方法。如需修改，请执行以下步骤：

1.  在知识库详情页，在文件列表中，找到目标文件，点击对应“操作”列中的设置按钮。
    
    ![](images/013.jpg)
    
2.  在“设置切片方法”页面，选择需要使用的切片方法。
    

### 删除文件/文件夹

在知识库中删除文件/文件夹时，仅从目标知识库中移除了该文件，**并非从系统中删除了该文件**，不会影响该文件在其他知识库中的正常使用。

1.  在知识库详情页，在文件列表中，找到目标文件，点击对应“操作”列中的删除按钮。
    
    ![](images/014.jpg)
    
    如需同时删除多个文档，可勾选目标文件，点击列表右上方的“批量删除”按钮。
    
2.  确认信息无误后，点击“确认删除”。
    

## 编辑知识库信息

1.  在“知识库管理”页面，将鼠标悬停在目标卡片上，点击卡片右上角的修改按钮。
    
    ![](images/143.jpg)
    
2.  在“编辑知识库”页面，更新知识库信息，包含名称、描述、默认切片方法等，点击“保存修改”。

    ![](images/144.jpg)

## 删除知识库

知识库删除后，引用该知识库的 Agent 将无法正常工作，请谨慎执行此操作。

1.  在“知识库管理”页面，鼠标悬停至目标知识库卡片，点击右上角的删除按钮。
    
    ![](images/142.jpg)
    
2.  确认删除信息无误后，点击“确认”。
